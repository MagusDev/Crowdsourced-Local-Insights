User Resource Profile

This profile describes the user resource in the GeoData API. A user represents a registered account that can post insights and give feedback on others' content. This profile includes all supported attributes and hypermedia controls.

Attributes

Attribute Type Required Description
usernamestringYesUnique user identifier
emailstringYesEmail address
phonenumberNoPhone number
passwordstringYesPassword (write-only)
first_namestringYesFirst name
last_namestringNoLast name
statusenumNoUser status: ACTIVE, INACTIVE, BANNED
roleenumNoUser role: USER, ADMIN
profile_picturestringNoBase64-encoded full image
profile_picture_thumbstringNoThumbnail image (auto-generated)
created_datedatetimeNoAccount creation timestamp
modified_datedatetimeNoLast modification timestamp

Supported Actions

  • GET — Retrieve user information
  • POST — Register a new user
  • PUT — Update user information
  • DELETE — Remove user account

Hypermedia Controls

Control Name Method Description
selfGETLink to this user resource
editPUTEdit user data
deleteDELETEDelete this user
profileGETLink to this profile document
geometa:insightcollectionGETList of all insights
geometa:user-feedbackcollectionGETFeedback submitted by the user
geometa:add-userPOSTCreate a new user
geometa:usercollectionGET(Admin only) List of all users
